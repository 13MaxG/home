I want to present few images from my last nootebook. It seems quite interesting stuff.

The function

This function produces image of specific spiral. I used that spiral in many places (for example here http://13maxg.tumblr.com/post/121924282647/sunflower-created-by-voronoi-diagram-of-special)
seq - sequence of numbers, Prime for example
from, to - range of indexes
alpha -  produces angle between two next points
func - function of distance

DrawFlower[seq_, from_, to_, alpha_, func_] := Module[{special}, special = Table[ { func[seq[i]] * Sin[2 * Pi/alpha * seq[i]], func[seq[i]] * Cos[2 * Pi/alpha * seq[i]]}, {i, from, to, 1}] ; Show[ Graphics[{PointSize[0.005], Blue, Point[special]}] ] ]

Examples

Plot every number from 1 to 1000, with angle parameter 34.

DrawFlower[# &, 1, 1000, 34, Sqrt]

Plot every number from 1 to 1000 with Golden Ratio as angle parameter

DrawFlower[# &, 1, 1000, GoldenRatio, Sqrt]

Same but with   3^(1/2)

DrawFlower[# &, 1, 1000, Sqrt[3], Sqrt]

Plot  squares of numbers between 1 and 1000

DrawFlower[#^2&, 1, 1000, GoldenRatio, Sqrt]

Plot prime numbers

DrawFlower[Prime, 1, 1000, GoldenRatio, Sqrt]
